Day-02 语言元素

Python的变量和类型

整型：Python3中整数只有int这一种了，且支持二进制，八进制，十进制，十六进制的表示法

浮点型：小数

字符串型：单引号或双引号括起来的任意文本，字符串还有原始字符串表示法，字节字符串表示法，Unicode字符串表示法，而且可以书写成多行的形式

布尔型：只有True和False两种值， 可以直接用True和False来表示布尔值（注意大小写）， 也可以通过布尔运算计算出来（3<5会产生布尔值True）

复数型：形如3+5j，跟数学上的复数表示一样，唯一不同的是虚部的i换成了j。

  • 变量命名硬性要求
    • 变量名有字母数字下划线构成，且数字不能开头
    • 大小写敏感
    • 不能和关键字冲突
  • 在对变量类型进行转换时可以使用Python的内置函数
    • int()：将一个数值或字符串转换成整数，可以指定进制。
    • float()：将一个字符串转换成浮点数。
    • str()：将指定的对象转换成字符串形式，可以指定编码。
    • chr()：将整数转换成该编码对应的字符串（一个字符）。
    • ord()：将字符串（一个字符）转换成对应的编码（整数）。
