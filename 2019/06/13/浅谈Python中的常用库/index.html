requests 库

快速上手

安装 requests 模块： pip install requests

导入 requests 模块：import requests


尝试获取某个网页：
GET: r = requests.get('http://www.xxx.com')
POST: r = requests.post('http://www.xxx.com/post', data={'key':'value'})
还有一些其他的请求

requests默认使用application/x-www-form-urlencoded对POST数据编码。


如何传递URL参数？如 http://httpbin.org/get?key1=value1&key2=value2&key2=value3
